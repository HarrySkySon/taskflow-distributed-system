### ЛР3 - Моделювання взаємодії
**Файл:** `ЛР3_Звіт_Моделювання_взаємодії.md`

**Містить PlantUML діаграми:**
1. **Sequence Diagram: Реєстрація користувача**
   - Синхронна взаємодія (REST API)
   - Асинхронна взаємодія (RabbitMQ events)
   - Взаємодія з БД та SMTP

2. **Sequence Diagram: Створення проєкту**
   - Автентифікація через Auth Middleware
   - Взаємодія між Projects Service та Users Service
   - Додавання членів команди

3. **Sequence Diagram: Створення задачі**
   - Публікація подій в RabbitMQ
   - Асинхронна обробка Notifications Service

4. **Схема обробки помилок**
   - Circuit Breaker pattern
   - Retry mechanisms
